## Cloning
``` bash
git clone https://github.com/marcio-pessoa/marcade.git
```

### Installing required Python modules
``` bash
pip3 install -r requirements.txt
```

## Running
Enter MArcade directory:
``` bash
cd marcade
```

To start a random game:
``` bash
./marcade.py
```

To start a game:
``` bash
./marcade.py invasion
```

For help:
``` bash
./marcade.py -h
```
